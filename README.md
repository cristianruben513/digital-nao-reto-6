# Digital NAO Reto 6

## Bases de datos no relacionales para almacenar datos en json

Para clonar el repositorio con la carpeta del servidor y la aplicacion movil:
```
git clone --recursive https://github.com/cristianruben513/digital-nao-reto-6.git
```

Para actualizar el repositorio con todos los submodulos incluidos:
```
git submodule update --init --recursive
```
